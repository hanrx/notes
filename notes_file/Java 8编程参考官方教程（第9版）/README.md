
# 目录/简要总结

## [第1章 Java的历史和演变](docs/第1章Java的历史和演变.md "第1章 Java的历史和演变")

* Java的历史和演变：
    * 成型动力及继承的思想：
        * 继承：继承其他编程语言最优元素。
        * 使命：为完成自身特殊使命所必需的创新性概念。
    * 语言创建与发展取决两个基本原因：
        * 适应环境和用途的变化。
        * 实现编程艺术的完善和提高。


* Java的家世：
    * 语言设计的每次革新，都是为了解决之前语言不能解决的基本问题。
    * Java从C继承了它的语法。
    * 面向对象特性则受C++的影响（C++是C的直接后代）。


* 现代编程语言的诞生》C语言：（迫切需求、和硬件的普及）
    * 影响：从根本上改变了编程的方式和思路。
    * 实现编程艺术：对结构化、高效率的高级语言需求直接结果。
    * 设计语言时，经常需要取舍，如：
        * 易用性与性能。
        * 安全性与效率。
        * 稳定性与可扩展性。
